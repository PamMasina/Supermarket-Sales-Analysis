# Supermarket-Sales-Analysis

Project Overview
This project involves the end-to-end processing of supermarket sales data, from raw CSV cleaning to interactive visualization.

Key Features:
Data Cleaning (Excel): Standardized date formats, corrected data types (Text to Number), and handled missing rating values.

Interactive Dashboard (Power BI): Created a dynamic report allowing users to filter sales by City, Product Category, and Time Period.

Insights: Identified Food and Beverages as the leading revenue generator and analyzed monthly sales trends using Power BI's date hierarchy.

Tools Used:
Excel: Data Cleaning & Pivot Tables

Power BI: Data Modeling & DAX Visualizations
